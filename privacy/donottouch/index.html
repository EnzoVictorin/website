Politique de confidentialité de
Don't Touch Your Screen

La protection de votre vie privée est très importante pour moi, c’est pourquoi je fais tout ce qui est en mon pouvoir pour minimiser les données collectées. L’application vous présente des annonces via le SDK Google Mobile Ads. Ce dernier collecte :

  • Votre adresse IP, qui est utilisée pour estimer l’emplacement général de l’appareil.
  • Les rapports de crash, qui sont utilisés pour améliorer le SDK Google Mobile Ads. Ces informations peuvent être utilisées à des fins publicitaires.
  • Des rapports de performances tels que:
    • Le temps de lancement de l’application
    • Taux de suspension.
    • L’utilisation de l’énergie.
    Ces données peuvent être utilisées pour évaluer le comportement des utilisateurs, comprendre l’efficacité d’une fonctionnalité existante et en planifier de nouvelles. Ils peuvent également être utilisés à des fins publicitaires.
  • Les rapports de performance des publicités qui contiennent ce que l’utilisateur a vu et des informations sur l’interaction avec les publicités.

Cette application ne suit pas votre activité sur d’autres applications et sites web. Aucune publicité personnalisée ne vous sera présentée. Si vous avez effectué l’achat in-app pour désactiver les annonces, le SDK Google Mobile Ads sera désactivé la prochaine fois que vous lancerez l’application.
Votre score est envoyé à Game Center pour vous placer dans un classement que tous les autres joueurs peuvent voir. Ce score est associé à un surnom que vous avez choisi lors de la configuration de Game Center. Le développeur ne collecte pas d’autres données.
